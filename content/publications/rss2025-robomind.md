RoboMIND dataset contains 107,000 demonstration trajectories covering 479 different tasks and 96 object categories, including multi-view observation data, robot proprioceptive state information, and language task descriptions. It is currently the largest and highest-quality multi-embodiment robot teleoperation dataset built on a unified platform.
